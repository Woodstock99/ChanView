# ChanView

ChanView erlaubt die Bearbeitung der Senderliste von LG Smart-TV's.

Ab etwa 2014 ist es möglich die Senderliste GlobalClone00001.TLL von LG-TV's mit webOS auf einen USB-Stick zu kopieren. Die Datei kann dann mit Hilfe von ChainView am Computer bearbeitet werden und anschließend wieder auf das TV-Gerät kopiert werden.

Die einzelnen Sender lassen sich editieren, sortieren, markieren, filtern, neu nummerieren oder auch vollständig löschen. Favoritenlisten können ebenfalls erstellt und bearbeitet werden. Darüber hinaus können auch die logischen Kanalnummern (LCN) in der Service-Info-Liste geändert oder komplett gelöscht werden.

ChainView wurde unter Verwendung eines LG-TV UM7100PLA entwickelt.
